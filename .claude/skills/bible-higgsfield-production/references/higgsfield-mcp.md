# Higgsfield MCP 제작 기준

## 모델 선택 원칙
- 기본 영상: `seedance_2_0`. 인물 정체성과 일반적인 움직임에 강한 기본값으로 사용한다.
- 대체 영상: `kling3_0`. 멀티샷, 강한 모션, 오디오 관련 옵션이 필요할 때 검토한다.
- 이미지 키프레임: 기본값은 GPT Image 2이며 모델 id는 `gpt_image_2`다. 세로 9:16 한국 웹툰형 구도를 명시한다.
- 모델 제약이 확실하지 않으면 먼저 모델 탐색 또는 비용 조회를 한다.

## 프롬프트 원칙
- 사용자-facing 문서와 생성 프롬프트는 한국어만 사용한다. 모델 id, JSON 키, 도구 이름 같은 기술 식별자는 예외다.
- 키프레임 프롬프트에는 "한국 웹툰형", "선명한 선화", "따뜻한 채색", "컷툰식 표정 연기", "세로 9:16"을 기본 스타일로 포함한다.
- 화면 속 글자가 필요한 경우 한국어만 허용하고, 영어/중국어/가짜 문자 생성 금지를 명시한다.
- 각 프롬프트에 피사체, 동작, 카메라, 렌즈감, 조명, 배경, 속도, 감정, 세로형 구도를 포함한다.
- 성경 구절 타이포그래피 구간은 인물 없는 그래픽 컷으로 분리하면 생성 안정성이 높다.
- 반복 인물은 동일한 의상, 헤어, 액세서리, 얼굴 특징을 매 컷에 반복한다.

## 비용 안전
- `get_cost: true` 또는 MCP 비용 조회를 먼저 실행한다.
- 비용 결과, 총 컷 수, 재시도 가능성을 사용자에게 보여준다.
- 명시 승인 전에는 생성하지 않는다.

## 실패 처리
- 인물 얼굴이 흔들리면 컷 수를 줄이거나 MCP 참조 이미지를 사용한다.
- 장소가 흔들리면 배경 디테일을 단순화한다.
- 대사가 긴 장면은 실제 입모양 싱크보다 자막과 반응 컷 중심으로 만든다.
- 공포 장면은 어둡게만 만들지 말고 모바일에서 보이는 명암을 유지한다.
