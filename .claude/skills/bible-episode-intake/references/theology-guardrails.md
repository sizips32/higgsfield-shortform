# 성경 쇼츠 신학/청소년 안전 가이드

## 기본 원칙
- "성경은 이렇게 말한다"와 "이 교단/수업에서는 이렇게 이해한다"를 구분한다.
- 한 영상에서 논쟁적 주장을 많이 넣지 않는다. 하나의 본문, 하나의 메시지, 하나의 적용으로 제한한다.
- 다른 교단 학생, 무신론자 친구, 의심하는 친구를 웃음거리로 만들지 않는다.
- 중3 대상 영상에서는 두려움보다 궁금증, 안심, 선택, 관계 회복을 우선한다.

## 죽음/귀신/부활 주제
- 사용할 수 있는 쉬운 표현: "성경은 죽음을 의식 없는 쉼, 잠에 비유해 설명하는 장면이 있어."
- 피할 표현: "너희가 아는 건 다 틀렸어", "귀신 믿으면 무식해", "죽으면 끝이야" 같은 공격적 단정.
- 권장 구조: 생활 속 공포 -> 성경 구절 -> 오해 정리 -> 현실 문제로 코믹 복귀.

## 구절 사용
- 구절은 짧게 화면 타이포그래피로 보여주고, 내레이션은 쉬운 말로 푼다.
- 본문 의미가 논쟁적이면 "이 관점에서는" 또는 "오늘은 이 구절을 중심으로"라고 범위를 좁힌다.

## 캐릭터 존중
- 재림교인 A: 성경 지식은 있지만 인간적인 약점이 있는 주인공.
- 무신론자 B: 논리적 질문을 던져 메시지를 선명하게 만드는 친구.
- 일요교회 C: 다른 배경을 가진 신앙인으로, 오해를 인정할 수 있지만 존중받아야 하는 친구.
