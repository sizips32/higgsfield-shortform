# 성경 쇼츠 시나리오 루브릭

## 길이 압축
- 60초 목표: 7~9컷, 대사 420~520자.
- 80~90초 목표: 9~12컷, 대사 600~800자.
- 100초 목표: 11~14컷, 대사 780~950자.
- 100초를 넘으면 농담, 설명, 반복 반응 순서로 줄인다.

## 리텐션 비트
- 0초: 시각적 문제 또는 감정 폭발.
- 8~12초: "왜?"가 생기는 정보.
- 25~35초: 오해 또는 반론.
- 45~55초: 성경 본문 제시.
- 65~80초: 현실 적용 또는 코미디 반전.

## 대사 톤
- 중3 친화: "헐", "인정", "잠깐만", "그럼 이건 뭐야?" 정도는 가능.
- 과한 유행어, 비하어, 교회 내부 은어는 피한다.
- 캐릭터별 말투는 다르되, 누구도 설명 기계가 되지 않게 한다.

## EP04-A 점검 요약
- 장점: 수행평가 위기라는 현실 훅, 귀신 전설과 성경 주제 연결, 타이포그래피 구절 구간, 코믹 엔딩이 좋다.
- 수정 필요: 1분40초는 요청 길이의 상한이므로, 초과하지 않도록 대사와 반복 반응만 정리한다.
- 수정 필요: 공동묘지/어르신 농담은 무덤과 죽음을 가볍게 다루는 느낌이 날 수 있어 "무서워할 대상이 아니라 조심스럽게 생각할 문제"로 완화한다.
- 수정 필요: C의 기존 믿음을 웃음거리로 만들지 말고 "나도 다르게 배웠는데, 본문 기준으로 다시 보자" 정도로 존중한다.
- 추천: 시편 146:4 한 구절만 중심으로 쓰고, 부가 구절은 설명 자료에만 남긴다.
