# 성경 쇼츠 큐레이션 루브릭

## 후보 평가 6축
1. 훅: 첫 2초에 멈춰 보게 하는가?
2. 명확성: 중3이 한 번에 이해하는가?
3. 정서: 불안보다 안심, 생각, 웃음, 공감으로 끝나는가?
4. 신학 안전성: 본문 의미를 과장하지 않는가?
5. 제작 품질: 얼굴, 손, 글자, 장소가 크게 깨지지 않는가?
6. 모바일 가독성: 자막, 표정, 행동이 작은 화면에서 보이는가?

각 축은 1~5점으로 평가한다. 총점은 참고용이며, 최종 선택은 사용자 판단을 우선한다.

## 제목 원칙
- 궁금증 + 주제 명확성: "밤 학교 귀신보다 무서운 것"
- 피할 제목: 특정 교단/사람을 조롱하는 제목, 과한 공포 제목, 본문을 왜곡하는 제목.

## 설명/댓글 원칙
- 설명은 핵심 구절과 질문 하나를 포함한다.
- 고정 댓글은 싸움을 유도하지 말고 생각을 여는 질문으로 쓴다.
