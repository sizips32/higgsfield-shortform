## 하네스: 중3 대상 성경 유튜브 쇼츠 제작

**목표:** 성경 주제, 시나리오, 또는 문서를 입력하면 중학교 3학년생이 이해하고 끝까지 볼 수 있는 1분~1분40초(60~100초) 세로형 유튜브 쇼츠를 기획, 제작, 큐레이션까지 진행한다.

**트리거:** 성경 쇼츠, 중학생 성경 영상, 교리 쇼츠, 유튜브 쇼츠 시나리오, Higgsfield 제작, 힉스필드 MCP 렌더, 또는 기존 성경 콘티 보완 요청 시 `bible-shorts-orchestrator` 스킬을 사용하라. 단순한 성경 질문이나 짧은 문장 교정은 직접 응답 가능하다.

**제작 고정값:** 키프레임 이미지는 기본적으로 GPT Image 2(`gpt_image_2`)를 사용하고, 결과물은 한국 웹툰형 세로 쇼츠로 만든다. 하네스 산출물과 생성 프롬프트는 한국어만 사용한다. 단, 모델 id, JSON 키, 도구 이름 같은 기술 식별자는 예외다.
